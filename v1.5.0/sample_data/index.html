Sample Data¶

Verde provides some sample data and ways of generating synthetic data through the verde.datasets module.

Where are my data files?¶

The sample data files are downloaded automatically by pooch the first time you load them. The files are saved to the default cache location on your operating system. The location varies depending on your system and configuration. We provide the verde.datasets.locate function if you need to find the data storage location on your system.

You can change the base data directory by setting the VERDE_DATA_DIR environment variable to the desired path.
